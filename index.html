WebEyeTrack for Qualtrics

Eye tracking calibration for survey research

Quick Start

Add eye tracking calibration to your Qualtrics survey:

  1. Create a Text/Graphic question in Qualtrics
  2. Switch to HTML view and paste the iframe code
  3. Done! Calibration data automatically saves to embedded data

See the Integration Guide for complete copy-paste code.
